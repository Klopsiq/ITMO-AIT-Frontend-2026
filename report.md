# Отчет по лабораторной работе №1

## Краткая документация по реализованным методам

### ConstructiveNumber

Реализован класс `ConstructiveNumber` для работы с конструктивными числами — парами $(a, b) \in \mathbb{Q}$, представляющими интервал $[a, b]$.

**Создание:**
1. Из пары чисел: `CN(a, b)` — задаёт интервал $[a, b]$
2. Из числа и точности: `CN(x, epsilon=ε)` — задаёт интервал $[x - \varepsilon/2, x + \varepsilon/2]$

**Операции:**
- Сложение, вычитание, умножение, деление — интервальная арифметика
- Все операции поддерживают как `CN + CN`, так и `CN + float` (и наоборот)
- Сравнение: `<`, `>`, `<=`, `>=`, `==`
- Возведение в целую степень
- Получение значения: `value(α)`, где $\alpha \in [0, 1]$: при $\alpha = 0$ → $a$, при $\alpha = 1$ → $b$
- Свойство `epsilon` — ширина интервала $\varepsilon = b - a$

### BlackBox (Чёрные ящики)

Реализованы три целевые функции с унифицированным интерфейсом:

1. **QuadraticGoodCond** — квадратичная 6-арная функция: $f(x) = \frac{1}{2} x^T A x + b^T x + c$, число обусловленности $\kappa \approx 1.16$
2. **QuadraticBadCond** — квадратичная 4-арная функция: $f(x) = \frac{1}{2} x^T A x + b^T x + c$, число обусловленности $\kappa \approx 100$
3. **Rosenbrock** — функция Розенброка 3-арная: $f(x) = \sum_{i=0}^{1} [100(x_{i+1} - x_i^2)^2 + (1 - x_i)^2]$

Каждый ящик предоставляет:
- `func(x)` — значение функции
- `grad(x)` — градиент
- `hess(x)` — гессиан
- Счётчики вызовов: `func_calls`, `grad_calls`, `hess_calls`
- Все функции совместимы с `ConstructiveNumber`

### Методы оптимизации

1. **Нелдер-Мид** (0-го порядка) — метод деформируемого многогранника. Не требует производных.
2. **Градиентный спуск** (1-го порядка) — классический градиентный спуск с фиксированным learning rate.
3. **Метод Ньютона** (2-го порядка) — использует гессиан для вычисления оптимального направления спуска.

## Общий вывод

1. **Метод Ньютона** — наиболее эффективен по числу итераций. Для квадратичных функций сходится за 1 шаг. Для Розенброка демонстрирует квадратичную скорость сходимости. Однако требует вычисления гессиана.

2. **Градиентный спуск** — скорость сходимости обратно пропорциональна числу обусловленности. При $\kappa \approx 1$ сходится быстро, при $\kappa \approx 100$ — значительно медленнее. Критически зависит от выбора lr.

3. **Нелдер-Мид** — надёжен и не требует производных, но сходится существенно медленнее, особенно в высоких размерностях.

4. **Динамика ε** — интервальная арифметика неизбежно расширяет ε с каждой операцией. Методы, сходящиеся за меньшее число итераций (Ньютон), накапливают меньшую погрешность. Рост ε зависит от числа обусловленности задачи.
